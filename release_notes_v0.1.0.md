# Locale v0.1.0

Initial release of **Locale** – a local, privacy-focused translation app powered by TranslateGemma 12B via Ollama.

## Highlights

- **120+ languages** with searchable dropdown (search by name or code)
- **Modern UI** – dark-first theme, emerald accents, Plus Jakarta Sans
- **Live connection status** – updates every 30s and when you return to the app
- **100% local** – no API keys, no cloud, no tracking

## Requirements

- [Ollama](https://ollama.com) installed
- TranslateGemma 12B model: `ollama run translategemma:12b`
- 16GB+ RAM recommended (8GB with 4B model)

## Quick Start

1. Install Ollama and the model (see above)
2. Download and run Locale
3. Connect and translate

See the [README](https://github.com/PierrunoYT/localtranslate) for full setup instructions.
